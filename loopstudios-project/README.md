# Loopstudios Landing Page

Landing page desarrollada con **Astro** y **Tailwind CSS**.

## 🚀 Cómo correr el proyecto

### 1. Instalar dependencias
```bash
npm install
```

### 2. Correr en modo desarrollo
```bash
npm run dev
```

### 3. Abrir en el navegador
```
http://localhost:4321
```

---

## 📁 Estructura del proyecto

```
src/
├── components/
│   ├── Header.astro      ← Navbar con menú móvil
│   ├── Hero.astro        ← Sección principal con imagen
│   ├── About.astro       ← Sección Interactive VR
│   ├── Creations.astro   ← Galería de creaciones
│   └── Footer.astro      ← Pie de página
├── layouts/
│   └── Layout.astro      ← Layout base HTML
├── pages/
│   └── index.astro       ← Página principal
└── styles/
    └── global.css        ← Estilos globales + Tailwind
public/
└── images/               ← Imágenes del proyecto
```

## 🎨 Tecnologías
- [Astro 6](https://astro.build/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- Fuentes: Alata + Josefin Sans (Google Fonts)
